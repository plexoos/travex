3 This package provides a library of C++ classes which can be used to evaluate and
4 validate reconstruction of charged particle tracks measured by tracking
5 detectors. One of the objective of this library is to design and implement such
6 functionality in a detector-independent way that can be applied to most common
7 detector configurations and reconstruction algorithms and frameworks used in
8 particle physics experiments.
9 
10 We use `doxygen` to document the code: <http://plexoos.github.io/travex/>
