5 This package provides a library of C++ classes which can be used to evaluate and
6 validate reconstruction of charged particle tracks measured by a tracking
7 detector. One of the objectives of this library is to design and implement such
8 functionality in a detector-independent way that can be applied to most common
9 detector configurations, reconstruction algorithms, and frameworks used in
10 particle physics experiments.
11 
12 The code and coding style in this project are heavily based on and influenced by
13 the [ROOT](http://root.cern.ch) framework. The source is available at
14 <https://github.com/plexoos/travex> and the `doxygen` generated documentation
15 can be found at <http://plexoos.github.io/travex>

18 The Concept
19 ===========
20 
21 This tools is designed to answer the following questions:
22 
23 - Given a hit produced and reconstructed in the detector (i.e. the hit is
24  available for reconstruction), what is the probability to assign this hit to
25  a reconstructed track?
26 
27 - Given a hit assigned to the reconstructed track, what is the probability for
28  this hit to be the actual (i.e. true) hit produced by the track?
29 
30 
31 The following entities must be provided by the user:
32 
33 - A reconstructed track
34 - A set of hit candidates for the track node in question
35 - A condition to select the best candidate
36 - Info what hit was accepted by the reconstruction algorithm (in case of not skipped layer)

40 How to build and install the library
41 ====================================
42 
43 
44 Prerequisites
45 -------------
46 
47 - C++ compiler with C++11 support (e.g. g++ >= 4.8.2)
48 - ROOT (>= 5.34.30), http://root.cern.ch
49 - boost libraries (>= 1.54): `program_options`, `regex`, and `filesystem`
50 
51 
52 Build with cmake
53 ----------------
54 
55 Checkout the code using the following command:
56 
57  git clone https://github.com/plexoos/travex.git
58 
59 Compile and build the library and the tools:
60 
61  cd travex/
62  mkdir build && cd build
63  cmake ../
64  make install
